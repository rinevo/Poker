# Poker Web App

シンプルな 5 枚ドロー・ポーカーのブラウザアプリです。

## 構成

- `index.html` - アプリ本体の HTML
- `styles.css` - アプリのスタイル
- `script.js` - ポーカーのロジックとインタラクション
- `cards/` - カード画像ファイル

## 使い方

1. `index.html` をブラウザで開きます。
2. 「配る」ボタンをクリックして手札を配ります。
3. カードをクリックしてホールド（保持）できます。
4. 「ドロー」ボタンをクリックしてホールドしていないカードを交換します。
5. `reset` ボタンでゲームをリセットします。

## 注意点

- `cards/` フォルダには `spades_a.svg` のようなカード画像が含まれている必要があります。
- `script.js` は `index.html` から読み込まれます。
